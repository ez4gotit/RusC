# RusLang для Visual Studio Code

Официальное расширение редактора для самого русского и суверенного языка
программирования.

## Возможности

- подсветка `.rus`, включая древнерусские синонимы и объектные роды;
- автоотступы и сворачивание блоков;
- переключение комментариев через `#`;
- сниппеты программ, условий, циклов, родов, зачинов и умений;
- структура документа для родов, черт и умений;
- краткие подсказки при наведении;
- диагностика `rusc` в панели Problems при открытии и сохранении;
- сборка, запуск, раскрытие C# и проверка автономных EXE;
- кнопка сборки в строке состояния.

## Компилятор

Расширение ищет `rusc.exe`:

1. по настройке `ruslang.compilerPath`;
2. в `artifacts/rusc/win-x64/rusc.exe` открытого проекта;
3. как команду `rusc` из `PATH`.

Сам компилятор в VSIX не встраивается: его версия и расширение редактора могут
обновляться независимо.

## Команды

- `RusLang: Собрать программу`;
- `RusLang: Запустить программу`;
- `RusLang: Раскрыть сгенерированный C#`;
- `RusLang: Проверить автономный EXE`;
- `RusLang: Показать здравие компилятора`.

## Установка VSIX

```powershell
code --install-extension ruslang-0.1.0.vsix
```
